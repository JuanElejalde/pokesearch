# 🔍 PokéSearch

> **by Juan Elejalde**

Buscador de Pokémon con datos en tiempo real de la PokeAPI, construido con JavaScript vanilla.

## 🛠️ Tecnologías

- JavaScript ES6+ (fetch, async/await, manejo de errores)
- HTML5 + CSS3
- PokeAPI (gratuita, sin autenticación)

## ✨ Funcionalidades

- Búsqueda por nombre o número de Pokémon
- Imagen oficial del Pokémon
- Tipos con colores distintivos
- Estadísticas base con barras de progreso
- Habilidades
- Manejo de errores (Pokémon no encontrado, fallo de conexión)
- Chips de ejemplo para búsqueda rápida

## 📁 Estructura

pokesearch/
├── index.html
├── style.css
└── app.js

## 🔑 API utilizada

- **PokeAPI:** https://pokeapi.co
- Gratuita · Sin API key · Sin límite de requests
